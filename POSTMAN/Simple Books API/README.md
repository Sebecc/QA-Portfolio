# 📚 Simple Books API Testing

## Project description

This repository contains manual API tests performed on the Simple Books API using Postman.

The goal of this project is to demonstrate practical REST API testing skills, including:

- API documentation analysis
- Creating and executing HTTP requests
- Response validation
- Positive and negative testing
- Writing test cases
- Bug analysis
- Working with Postman collections
